# Проверка гипотез и анализ результатов A/B-теста

## Цель проекта
Провести статистическую проверку гипотез и оценить влияние изменений продукта на пользовательское поведение и бизнес-метрики.

# Часть 1: Проверка статистической гипотезы

## Контекст
Проверяется гипотеза о различии пользовательской активности между двумя городами.

## Гипотеза
- H0: среднее время, проведённое в приложении пользователями из Москвы и Санкт-Петербурга, не отличается  
- H1: пользователи из Санкт-Петербурга проводят больше времени в приложении  

## Подход
- Проведена предобработка данных и удаление аномальных значений  
- Использован t-тест с поправкой Уэлча  
- Уровень значимости: 0.05  

## Результаты
- p-value = 0.9  
- Статистически значимых различий между группами не обнаружено  

## Вывод
Средняя активность пользователей из Москвы и Санкт-Петербурга статистически не различается.

## Интерпретация
Отсутствие различий может быть связано с:
- схожей аудиторией пользователей  
- одинаковыми маркетинговыми условиями  

# Часть 2: Анализ A/B-теста (интернет-магазин BitMotion Kit)

## Контекст
Онлайн-магазин протестировал новую версию интерфейса с целью увеличения конверсии в покупку.

## Гипотезы
- H0: изменение интерфейса не влияет на конверсию  
- H1: изменение интерфейса увеличивает конверсию  

## Метрика
- Конверсия в покупку (CR)

## Подход
- Проверена корректность разбиения пользователей на группы  
- Рассчитана конверсия в покупку по группам  
- Для проверки гипотез использован z-тест для сравнения долей
- Уровень значимости: 0.05

## Результаты
- Конверсия увеличилась с 28% до 30%  
- p-value = 0.028 < 0.05  
- Различия статистически значимы  

## Вывод
Изменение интерфейса статистически значимо увеличивает конверсию пользователей.

## Бизнес-рекомендация
Рекомендуется внедрить новую версию интерфейса для всех пользователей, так как это приводит к статистически значимому росту конверсии.
