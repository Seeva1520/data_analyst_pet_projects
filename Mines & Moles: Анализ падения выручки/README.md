# Mines & Moles: Анализ падения выручки

## Описание проекта
Анализ причин снижения выручки в мобильной игре Mines & Moles после июльского обновления 2024 года. 
Проект демонстрирует навыки продуктовой аналитики: построение дерева метрик, формулирование и проверку гипотез, 
анализ воронки конверсии, когортный анализ и визуализацию данных.

## Проблема
С июля 2024 года команда зафиксировала стагнацию, а затем падение выручки внутриигрового магазина. 
Предполагаемые причины: изменение алгоритма генерации уровней и редизайн магазина.

## Ход анализа
1. Построено дерево метрик для декомпозиции выручки.
2. Сформулированы две основные гипотезы о причинах падения.
3. Проведен анализ:
   - Активной аудитории (MAU, New Users, Retention)
   - Качества пользовательского опыта (доля пройденных уровней)
   - Монетизации (ARPAU, конверсия в покупку, средний чек)
4. Выявлены ключевые точки отказа: редизайн магазина разрушил воронку продаж, новый алгоритм повысил сложность уровней, что снизило удержание.

## Ключевые выводы
- **MAU** продолжает расти, но темпы замедлились (не основная причина падения выручки).
- **Retention Rate** и **доля прохождения 10+ уровней** резко упали после внедрения нового алгоритма генерации уровней.
- **ARPAU** снизился из-за критического падения **конверсии в покупку** после редизайна магазина.
- **Средний чек** и его компоненты (цена предметов, количество в заказе) остались стабильны.

## Рекомендации
1. Провести A/B тест для изолированной оценки влияния каждого из двух изменений.
2. Откатить редизайн магазина или доработать воронку на основе анализа поведения пользователей.
3. Сбалансировать сложность уровней нового алгоритма, чтобы снизить порог входа для новых игроков.
